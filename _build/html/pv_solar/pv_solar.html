Solar tools

Solar position

The solar position is based on the work of Reda & Andreas [RA08] which includes the Meeus [Mee98] and Iqbal [Iqb83] algorithms. For more details on the solar position algorithm, refer to Reda and Andreas. The algorithm use the earth Δt and Universal Time (UTC) fro time correction. For more information on Δt and Universal Time, see Nasa.

Bird’s Clear Sky

Brird’s clear sky irradiation model is based on work of Bird [Bir81]. For more details on the Bird’s algorithm, refer to Bird.

Climatology

The probabilistic forecasts in this package use Clear Sky Indexes \(CSI\) and clear sky data \(GHI_{\text{Clear Sky}}\).

\[CSI=\frac{GHI}{GHI_{\text{Clear Sky}}}\]
The climatology can be computed in two ways:
  1. A distribution is generated from all clear sky indices (\(\phi=ecdf(CSI)\)).

    The forecast is generated based on this single empirical distribution.

  2. For each time slot, an empirical distribution (ecdf) is generated (\(\phi_t=ecdf(CSI_t)\)).

    For example, if there are 10 hours in a day, 10 CSI distributions will be generated. Forecasts for each time range will be generated by its own distribution.

Note

CSI data is filtered with two main parameters:
  1. The zenith angle, all data corresponding to angles greater than this value will be ignored.

    The default value is set to 86°

  2. The maximum CSI value: Values greater than \(CSI_{max}\) will be replaced by \(CSI_{max}\).

    The maximum value is calculated as the minimum of the user-defined (\(CSI_{\text{user}}\)) and the fraction (\(f_0 \leq 1\)) of extraterrestrial solar radiation (\(E_0=1367 Wm^{-2}\)) reaching the Earth’s surface divided by the max value of the global horizontal irradiation \(GHI\).

    \[CSI_{max}=min\left(CSI_{user},\frac{E_0 \times f_0}{max({GHI})}\right)\]

The climatology computation rely on solar.ecdf0()
